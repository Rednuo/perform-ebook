# HTML/CSS

看一看最最基础和简单的的HTML/CSS，之所以从它入手，是因为所见即所得，你不需要配置任何的环境，只需要一个浏览器和一个编辑器。

任何事情，先从 `Hello World` 开始（不要问我为什么，推荐你使用Chrome浏览器和Sublime Text。）：

```Shell
cd ~/
mkdir demo
cd demo
touch index.html
```

输入如上的命令创建一个`index.html`文件。

```HTML
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title></title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>
```
